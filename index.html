Since I do not understand how websites work entirely yet, as a starting point, I have decided to take a course on Coursera called "HTML, CSS, and Javascript for Web Developers" by Yaakov Chaikin of JHU (https://www.coursera.org/learn/html-css-javascript-for-web-developers/home/welcome)

It is a 5 week course and involves building a restaurant site as a project. Apart from buiilding the restaurant site as an assignment, I have decided to use this as my personal webpage to explore and tweak around with tools I learn from the course as a more hands-on learning process.

Each week of the course corresponds to a separate module of the syllabus, which are:

1. Introduction to HTML5
2. Introduction to CSS3
3. Coding the Static Restaurant Site
4. Introduction to Javascript
5. Using Javascript to Build Web Applications

As I take the course sequentially, I will be using lessons learnt in each session and apply them to upgrade this website, which I feel would give me maximum benefit out of the lesson structure.
